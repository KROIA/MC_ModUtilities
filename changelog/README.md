# MC_ModUtilities — Changelog

A shared utility library for Minecraft 1.21.1 mods, providing reusable systems for GUI, networking, persistence, settings, events, and in-game testing. Built on Architectury API to support Fabric, NeoForge, and Quilt from a single codebase.

See the [documentation](../documentation/README.md) for API guides and usage examples.
