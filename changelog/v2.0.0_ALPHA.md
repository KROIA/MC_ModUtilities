# 2.0.0_ALPHA — In Development

Minecraft 1.21.1 | Fabric, NeoForge, Quilt

Major release: in-game test suite framework, comprehensive documentation, 50+ bug fixes across networking, GUI, persistence, and platform compatibility.

## Security

| Tag | Change |
|---:|---|
| ![hardening] | Shared secret comparison now uses `MessageDigest.isEqual` for constant-time comparison, preventing timing attacks against multi-server authentication |
| ![hardening] | Codec bounds checks on array length, element count, indices, and enum ordinals — crafted packets now throw `DecoderException` instead of corrupting state |
