# 2.0.0_ALPHA — In Development

Minecraft 1.21.1 | Fabric, NeoForge, Quilt

Major release: in-game test suite framework, comprehensive documentation, 80+ bug fixes across networking, GUI, persistence, and platform compatibility. 142 in-game unit tests across 7 suites.

## Security

| Tag | Change |
|---:|---|
| ![hardening] | Shared secret comparison now uses `MessageDigest.isEqual` for constant-time comparison, preventing timing attacks against multi-server authentication |
| ![hardening] | Codec bounds checks on array length, element count, indices, and enum ordinals — crafted packets now throw `DecoderException` instead of corrupting state |

## Performance

| Tag | Change |
|---:|---|
| ![bugfix] | `DataPersistence.saveDataCompoundList` — pre-computes element sizes for O(N) chunking instead of O(N²) full-chunk reserializations |
| ![improvement] | `Signal` / `DataEvent` — replaced immutable `Pair` listener storage with mutable `ListenerEntry`; eliminates O(n²) `CopyOnWriteArrayList.set()` during dispatch |
