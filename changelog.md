## Changelog Format

Each version changelog in the `changelog/` directory includes:

- **Version Information** - Version number, Minecraft version, release date
- **Summary** - High-level overview of changes
- **New Features** - Major features added
- **Improvements** - Enhancements to existing features
- **Bug Fixes** - Issues resolved
- **Documentation** - Documentation updates
- **API Changes** - Breaking changes and migrations
- **Dependencies** - Dependency version updates
- **Known Issues** - Outstanding issues and limitations
